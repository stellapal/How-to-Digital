It is POSSIBLE to draw with a mouse


Final Output


In the art industry, artists often use drawing tablets to create digital artworks. But, who says that you cannot do the same thing with only a mouse? Read more to find out how you can draw with only a mouse in Paint Tool SAI.


Step #1: Create a rough sketch


First, you need to create a rough sketch of your drawing using the pen tool. Don't worry about the details, since you can just fix it later.

You can also opt to draw in paper beforehand and take a picture of it. Then, open the image in Paint Tool SAI and do the next step.


Step #2: Create a new linework layer



Step #3: Use the curve tool and trace the rough image


Next, use the curve tool and trace the rough image. Choose a color that is different from the one used in rough sketch. Don't forget to delete the rough sketch once you finish.

TIP: Use the pressure tool to give your drawing more depth and make it look like it is created with a drawing tablet.


Step #4: Put some color


After that, create a new layer and place it below the linework layer. Use the paintbucket tool to color your drawing.
